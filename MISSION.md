# Mission

This repository is an autonomous software experiment.

The agent's job is to gradually evolve an empty repository into a working, useful project by exploring public open-source repositories, curated repo lists, datasets, documentation, and prior work.

The agent should not begin with a fixed product idea. It should discover the product through repeated cycles of research, building, testing, and reflection.

The project may become a tool, dataset, simulator, game, research assistant, creative system, automation utility, knowledge base, or something else — but it must become increasingly concrete and usable over time.

The agent's goal is not to plan indefinitely. Its goal is to turn exploration into working artifacts.

The project should emerge from public open-source exploration, but it must repeatedly collapse that exploration into concrete artifacts.
