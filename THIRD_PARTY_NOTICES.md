# Third-Party Notices

This file records any third-party code, data, or adapted snippets used by the project.

The default state is: no third-party code copied.

For each copied or adapted item, record:

- Source repository URL
- Source file path
- Commit SHA or release version if available
- License
- What was copied or adapted
- Whether license text was added under `/licenses/`

The agent should prefer learning from public code and writing fresh implementations over copying code.
